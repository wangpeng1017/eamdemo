# 慧新 EAM 系统

企业资产管理系统 Demo

## 功能模块

- 设备台账
- 维修管理
- 备品备件
- 维护保养

## 技术栈

- Next.js 16
- React 19
- Ant Design 6
- TypeScript
- Tailwind CSS 4

## 部署

项目已部署到 Vercel: https://eamdemo.aifly.me

## 访问

直接访问首页会自动重定向到管理后台仪表盘：
- 首页：https://eamdemo.aifly.me
- 控制台：https://eamdemo.aifly.me/admin/dashboard
- 设备台账：https://eamdemo.aifly.me/admin/equipment
- 维修管理：https://eamdemo.aifly.me/admin/repair
- 备品备件：https://eamdemo.aifly.me/admin/spareparts
- 维护保养：https://eamdemo.aifly.me/admin/maintenance
