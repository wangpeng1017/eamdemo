# entrustment 委托管理模块

委托业务全流程管理：咨询登记 → 报价 → 合同 → 委托单
⚠️ 文件夹变化时请更新此文件

## 文件清单

| 文件夹 | 地位 | 功能 |
|--------|------|------|
| client/ | 支撑 | 客户管理 |
| consultation/ | 入口 | 业务咨询登记 |
| quotation/ | 核心 | 检测报价管理，生成合同 |
| contract/ | 核心 | 检测合同管理，生成委托单、PDF |
| list/ | 核心 | 检测委托单管理，分配检测任务 |

## 业务流程

```
客户咨询 → 生成报价 → 签订合同 → 创建委托单 → 分配检测任务
consultation → quotation → contract → list → task
```

## 操作列规范

所有页面操作列遵循统一规范：
- **业务按钮**（左侧）：带文字+图标，如"生成PDF"、"生成委托单"
- **通用按钮**（右侧）：仅图标，查看/编辑/删除
- **宽度**：自适应（`whiteSpace: 'nowrap'`），不设固定宽度
